Projetos

Totem de atendimento em C

Neste projeto, tivemos como objetivo recriar um totem de autoatendimento de fast food utilizando a linguagem C. Aplicamos os conhecimentos básicos da linguagem para desenvolver uma solução funcional, priorizando a lógica de programação, a experiência do usuário e a organização do código para alcançar o melhor resultado possível.

Urna eletronica em Java

Neste projeto, avançamos um pouco mais no nível de programação, com o objetivo de desenvolver uma réplica funcional de uma urna eletrônica. A proposta incluiu não apenas a criação da interface e da lógica de votação, mas também a simulação de um partido político para a realização de uma eleição interna. O projeto permitiu aplicar conceitos como estruturas condicionais, manipulação de dados e interação com o usuário, aproximando-nos de um cenário real de aplicação da tecnologia em processos democráticos.

Restaurante HTML e CSS

Neste projeto, iniciamos nossos estudos em front-end com a criação do nosso primeiro site. A proposta teve como temática algo totalmente lúdico e fruto da nossa imaginação, permitindo explorar a criatividade enquanto aplicávamos conceitos básicos de HTML, CSS e estruturação de páginas web.
